# README

Commands use `make`.

| Command | Description                                    |
| ------- | ---------------------------------------------- |
| `build` | Builds the site in the `docs` directory.       |
| `clean` | Removes untagged Docker images.                |
| `help`  | Displays a list of all the available commands. |
| `image` | Builds the Docker image.                       |
| `shell` | Enters the container's bash shell.             |
